# Habit Tracker API

API REST para gestión de hábitos y metas personales.

## Tecnologías
- Java 17
- Spring Boot 3.5.12
- PostgreSQL
- JPA / Hibernate

## Endpoints

### Usuarios
- GET /usuarios
- GET /usuarios/{id}
- POST /usuarios
- PUT /usuarios/{id}
- DELETE /usuarios/{id}

### Metas
- GET /meta
- POST /meta
- PUT /meta/{id}
- DELETE /meta/{id}

### Hábitos
- GET /habitos
- POST /habitos
- PUT /habitos/{id}
- DELETE /habitos/{id}
